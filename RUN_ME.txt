OSINT Engine v4.2 — real CORS fix

1. Install Node.js 18 or newer.
2. Put these files in the same folder:
   - osint_engine_v4-2_CORS_FIXED.html
   - server.js
   - package.json
3. Open a terminal in that folder.
4. Run:
   npm start
5. Open:
   http://localhost:8787/osint_engine_v4-2_CORS_FIXED.html

Do NOT open the HTML directly with file:// if you want proper external API calls.
The browser cannot bypass CORS by itself. The local backend proxy is the correct fix.
